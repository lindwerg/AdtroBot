# BotFather Setup

Тексты для настройки бота через @BotFather.

## /setdescription (512 символов max)

Копировать и вставить в BotFather после команды /setdescription:

```
Астрологический бот с AI — ежедневные гороскопы, расклады таро, натальные карты.

Бесплатно:
• Гороскоп на день для всех знаков
• Карта таро дня
• 1 расклад таро в день

Premium (299 руб/мес):
• Персональный гороскоп по натальной карте
• 20 раскладов таро в день
• Кельтский крест (10 карт)
• Полная натальная карта с интерпретацией
```

## /setabouttext (120 символов max)

Копировать и вставить в BotFather после команды /setabouttext:

```
AI гороскопы и таро. Персональная астрология по натальной карте.
```

## /setcommands

Копировать и вставить в BotFather после команды /setcommands:

```
start - Главное меню
help - Справка и FAQ
about - О боте
```

## Как настроить

1. Открыть @BotFather в Telegram
2. Выбрать своего бота через /mybots
3. Нажать "Edit Bot"
4. Выполнить каждую команду и вставить соответствующий текст

## Keywords для поиска

Description содержит ключевые слова для поиска в Telegram:
- гороскоп, гороскопы
- таро, расклад
- астрология, астрологический
- натальная карта
