# Phase 8: Premium Tarot + Natal - Context

**Gathered:** 2026-01-23
**Status:** Ready for planning

<domain>
## Phase Boundary

Платные пользователи получают расширенные возможности таро и натальной карты:
1. **20 раскладов в день** (вместо 1 для бесплатных)
2. **Кельтский крест** — расклад на 10 карт с детальной AI интерпретацией
3. **История раскладов** — просмотр прошлых консультаций
4. **Натальная карта** — полная карта с позициями планет, домами, аспектами + AI интерпретация

Эта фаза расширяет функционал Phase 4 (Free Tarot) и Phase 7 (Premium Horoscopes), добавляя premium опыт для платных подписчиков.

</domain>

<decisions>
## Implementation Decisions

### Кельтский крест UI
- **Выбор типа расклада**: 2 отдельные кнопки в меню Таро — "Расклад 3 карты" и "Кельтский крест"
- **Ритуал**: Более длинный ритуал перед Кельтским крестом (серьёзнее, чем у 3-карточного)
- **Free user experience**: Premium тизер при нажатии "Кельтский крест" (описание + кнопка подписки)

### История раскладов
- **Точка входа**: Кнопка "История раскладов" в меню Таро
- **Формат списка**: Дата + вопрос + тип расклада (например: "Вчера | Карьера | 3 карты")

### Натальная карта display
- **Точка входа**: Отдельный раздел "Натальная карта" в главном меню (расширяем 2x2 до 2x3)
- **Формат вывода**: Текст (интерпретация) + изображение (визуализация карты)
- **Генерация изображения**: Самостоятельная генерация красивого идеального изображения (SVG через Python с кругом, планетами, знаками, аспектами)

### AI интерпретации
- **Кельтский крест (10 карт)**:
  - Объём: 800-1200 слов
  - Структура: По тематическим блокам (Ситуация → Препятствия → Ресурсы → Исход)
- **Натальная карта**:
  - Объём: 1000-1500 слов (максимальная детализация)

### Claude's Discretion
- Как показывать 10 карт Кельтского креста (последовательно vs album vs комбинация)
- Как пользователь открывает прошлый расклад (inline кнопки vs pagination)
- Сколько раскладов хранить в истории (лимит по времени или количеству)
- Структура AI интерпретации натальной карты (по разделам vs целиком)
- Стиль интерпретаций для premium контента (новичкам vs опытным vs адаптивный)

</decisions>

<specifics>
## Specific Ideas

- **Изображение натальной карты**: "Я хочу идеальное красивое изображение которое сгенерируем сами" — собственная генерация SVG, не внешние API
- **Кельтский крест**: Должен чувствоваться серьёзнее, чем 3-карточный расклад (более длинный ритуал, больше объём интерпретации)
- **История раскладов**: Пользователь должен видеть свой вопрос в списке, чтобы вспомнить контекст

</specifics>
