# Requirements: AdtroBot

**Defined:** 2026-01-22
**Core Value:** Качественная AI интерпретация астрологии и таро, которая конвертирует бесплатных пользователей в платных

## v1 Requirements

### Authentication & Onboarding

- [x] **AUTH-01**: Пользователь регистрируется автоматически при /start — ✓ Shipped in Phase 2
- [x] **AUTH-02**: Система автоматически определяет знак зодиака по дате рождения — ✓ Shipped in Phase 2
- [x] **AUTH-03**: Пользователь вводит дату рождения (для гороскопов и натальной карты) — ✓ Shipped in Phase 2
- [x] **AUTH-04**: Пользователь вводит время и место рождения (для натальной карты) — ✓ Shipped in Phase 7
- [x] **AUTH-05**: Пользователь получает immediate value (первый прогноз) сразу после регистрации — ✓ Shipped in Phase 2
- [x] **AUTH-06**: Данные пользователя сохраняются в БД (знак, дата/время/место рождения) — ✓ Shipped in Phase 2

- [x] **HORO-02**: Гороскоп доступен для всех 12 знаков зодиака — ✓ Shipped in Phase 3
- [x] **HORO-03**: Гороскоп красиво отформатирован с emoji и разметкой — ✓ Shipped in Phase 3
- [x] **HORO-04**: Платный пользователь получает детальный гороскоп по сферам (любовь, карьера, здоровье, финансы) — ✓ Shipped in Phase 7
- [x] **HORO-05**: Платный пользователь получает персональный прогноз на основе натальной карты — ✓ Shipped in Phase 7
- [x] **HORO-06**: Пользователь получает push-уведомление о новом ежедневном гороскопе — ✓ Shipped in Phase 3

- [x] **TAROT-02**: Бесплатный пользователь может сделать 1 расклад в день (3 карты: прошлое-настоящее-будущее) — ✓ Shipped in Phase 4
- [x] **TAROT-03**: Пользователь задаёт вопрос перед раскладом — ✓ Shipped in Phase 4
- [x] **TAROT-04**: Бот случайно выбирает карты из колоды (78 карт Райдер-Уэйт) — ✓ Shipped in Phase 4
- [x] **TAROT-05**: Бот учитывает прямые и перевёрнутые карты — ✓ Shipped in Phase 4
- [x] **TAROT-06**: Бот показывает изображения выбранных карт — ✓ Shipped in Phase 4
- [x] **TAROT-07**: Бот даёт AI интерпретацию расклада на основе вопроса и карт — ✓ Shipped in Phase 5
- [x] **TAROT-08**: Платный пользователь может делать 20 раскладов в день — ✓ Shipped in Phase 8

- [x] **TAROT-11**: Система лимитов отслеживает количество раскладов с атомарными операциями — ✓ Shipped in Phase 4

- [x] **NATAL-02**: Система рассчитывает позиции планет в домах и знаках (Swiss Ephemeris) — ✓ Shipped in Phase 8

## Out of Scope

Явно исключено, чтобы избежать scope creep:

| Feature | Reason |
|---------|--------|
| Мобильное приложение (iOS/Android) | Только Telegram бот — проще поддерживать и быстрее запускать |
| Живые тарологи/астрологи | Операционная сложность, фокус на AI качестве |
| Групповые расклады | Усложнение без очевидной ценности |
| Соцсети (Instagram, VK, Facebook) | Фокус на одной платформе (Telegram) |
| Обучающие курсы | Бот для прогнозов, не образовательная платформа |
| Геймификация с лидербордами | Отвлекает от core value |
| Чат между пользователями | Превращает бот в соцсеть |
